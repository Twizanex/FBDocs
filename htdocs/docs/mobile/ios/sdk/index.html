iOS SDK Reference

SDK Reference › iOS SDK Reference

The iOS SDK provides Facebook Platform support for iOS apps written in Objective-C. It enables you to access the Facebook Platform APIs including the Graph API, FQL, and Dialogs.

The SDK is open source and is available on GitHub. Read more in iOS Getting Started about how to set up the iOS SDK.
