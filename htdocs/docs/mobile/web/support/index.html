Mobile Web Feature Support

Below is a list of features that aren't currently supported on the mobile web. We'll update this page as this functionality becomes available.

Dialogs

  • Send dialog
  • Friend request dialog
  • Requests dialog parameters ('filters', 'exclude_ids', 'max_recipients', and 'data')
  • News Feed dialog parameters ('description' will not display in mobile News Feed, but it will display on desktop)
  • Sending a request to a non-friend
  • The 'display' parameter in all dialogs

Social Plugins

  • Send
  • Comments
  • Activity Feed
  • Recommendations
  • Like box
  • Facepile
  • Registration
  • Live stream

News Feed and Timeline

  • Usage stories in News Feed
  • Timeline units
  • News Feed action links
