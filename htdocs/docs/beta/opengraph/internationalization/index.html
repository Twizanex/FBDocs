Open Graph Internationalization

Core Concepts › Open Graph › Open Graph Internationalization

Open Graph supports internationalization (i18n) of actions and objects for use in multiple languages. Open Graph i18n follows the same process as the Facebook app i18n process, as outlined here. Once an app’s strings are in our system for translation, the developer can translate their strings or leverage the app's user community to help with translation. (Users can help translate using the Translations app.) Once translated, they must go through the same approval process as other translated strings (i.e. the language admins & owners agree that they are good translations).
