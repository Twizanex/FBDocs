Social Cafe

Download | Run

This sample demonstrate a simple app that publishes custom 'drink' actions for a coffee shop. NOTE: Currently, only administrators, developers, and testers of this application will be able to publish actions.



Wishlist Mobile

Download

This sample contains a wishlist application, with projects for iOS and Android. It demonstrates the use of custom objects and actions on mobile platforms.



Recipe of the Day

Download

Open Graph Tutorial sample app written in a single HTML page with the Facebook Javascript SDK. This simple sample creates a Recipe Object and Cook action for the user and alerts the user upon success. Also demonstrates the new Social Plugins: Add to Timeline and the Activity Feed plugins.
