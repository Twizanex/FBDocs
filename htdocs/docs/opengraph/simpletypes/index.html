Simple Types

Core Concepts › Open Graph › Simple Types

The Open Graph defines the following simple types for use in your custom object and action properties.

Type Description Literals
Boolean A Boolean represents a true or false value true, false, 1, 0
DateTime A DateTime represents a temporal value composed for a date (year, month, day) and an optional time component (hours, minutes).

Supports the ISO 8061 timestamp format with the timezone defaulting to UTC. Expected format is YYYY-MM-DDThh:mm:ssZ, where Z is an offset from UTC to express a different time zone. Examples:
  • January 26th, 2011 is 2011-01-26
  • January 26th, 2011 at 7:15pm UTC
    is 2011-01-26T19:15
  • January 26th, 2001 at 7:15pm PST
    is 2011-01-26T19:15-8:00
 Also supports literals that conform to the following patterns:

%Y%m%dT%H%MZ
%Y%m%dT%H%M
%Y-%m-%dT%H:%MZ
%Y-%m-%dT%H:%M
%Y-%m-%d
%d-%m-%Y
%Y/%m/%d
%m/%d/%Y
%Y/%m/%dT%H:%MZ
%Y/%m/%dT%H:%M
Enum A type consisting of bounded set of constant String values (enumeration members). Note: you can not current create properties of the Enum type A String value that is a member of the enumeration
Float A 64-bit signed floating point numeric value All literals that conform to the following formats:

1.234
-1.234
1.2e3
-1.2e3
7E-10
Integer A 32-bit signed integer numeric value All literals that conform to the following formats:

1234
-123
0123 (octal)
0x1A (hex)
String A sequence of UTF-8 characters All literals composed of UTF-8 characters with no escape characters
URL A sequence of UTF-8 characters that identify a Internet resource. Note: you can not current create properties of the URL type. All valid URLs that utilize the http:// or https:// protocols

Array

The Open Graph also supports an array type that represents a homogeneous, ordered sequence of simple, complex or object values. Use an array when you want a collection of values within one property. You define an array value by repeating the same meta tag name for each element of the array.

Example

<html>
    <head prefix="og: http://ogp.me/ns# 
         your-og-app: http://ogp.me/ns/apps/your-og-app#">
      <meta property="fb:app_id"                     content="YOUR_APP_ID"> 
      <meta property="og:type"                       content="your-og-app:SimpleObject"> 
      <meta property="og:title"                      content="Simple Object"> 
      <meta property="og:description"                content="A great simple object"> 
      <meta property="og:image"                      content="https://your-great-image">
      <meta property="og:url"                        content="https://this-url">
      <meta property="my-og-app:stringproperty"      content="A String value"> 
      <meta property="my-og-app:booleanproperty"     content="true"> 
      <meta property="my-og-app:datetimeproperty"    content="2011-09-22"> 
      <meta property="my-og-app:floatproperty"       content="42.1"> 
      <meta property="my-og-app:integerproperty"     content="123">
      <meta property="my-og-app:stringarrayproperty" content="String1">
      <meta property="my-og-app:stringarrayproperty" content="String2">
  </head>
  <body>
    <!-- Any content you want to display to the user -->
  </body>
</html>
