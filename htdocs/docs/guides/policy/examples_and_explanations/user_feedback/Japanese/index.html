ユーザーからの評価

はじめに

ユーザーからの評価は、あなたのアプリとそのコンテンツに対するユーザーとその友達の反応をより正確に理解するのに役立ちます。また、ライブアプリを自動または手動でプラットフォームに配置する際にも役立ちます。ユーザー体験を監視すれば、どのアプリがプラットフォームのエコシステムに良い影響または悪い影響を与えているかを判断できるからです。

開発者の連絡先メールアドレスに送信されたレポートを監視して返答したり、アプリの [インサイト]ページを頻繁に確認することをお勧めします。[インサイト]ツールでは、アプリの配信状況や、以下に説明する評価の多くを監視できます。

アプリに対する反応

ユーザーは、アプリの情報ページにあるようなリンクを使用して、アプリのブロック、アンインストール、報告といったさまざまな方法で、アプリに否定的な評価を下すことができます。

また、ユーザーは招待やリクエストを使ってアプリをブロックすることもできます。

アプリによって作成されるコンテンツに対する反応

ユーザーはアプリによって作成されるコンテンツに対し、多くの反応を示すことができます。

コンテンツに対するフィードバックとしては、記事をスパムとして非表示にしたりマークすることが可能ですが、これらに制限されるわけではありません。ニュースフィード記事の品質については常に念頭に置いてください。品質は記事を読む受信者やユーザーによって評価されるので、コンテンツに肯定的な評価を得られるように気をつけましょう。(プロフィールに公開されているコンテンツは、公開しているユーザーの友達のニュースフィードに表示されることがあります)。

写真やイベントなどのその他のチャンネルについては、フィードバックには写真からのタグの削除、写真の削除または報告、イベントの削除または報告などが含まれますが、これらに制限されるわけではありません。
