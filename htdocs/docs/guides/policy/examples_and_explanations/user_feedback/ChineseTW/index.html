用戶意見

簡介

這些訊號能助您更清楚了解用戶與其朋友對應用程式及其內容的評價。這些意見亦有助我們在開放平台上自動化及手動執行批准應用程式上線的決定，因為監控用戶體驗有助我們了解哪些應用程式向用戶提供了正面或負面的氣氛，影響平台氣氛。

我們鼓勵您監控並回應向您開發人員聯絡電郵的報告，並經常前往您應用程式的 精準行銷資料頁面。精準行銷工具讓您觀察應用程式的傳播及以下概述的多個訊號。

向應用程式作出的回應

用戶可透過多種方式提供應用程式的負面意見，例如封鎖、解除安裝或透過應用程式資訊頁面的連結等舉報應用程式。

用戶亦可透過發送的邀請/請求直接封鎖應用程式。

向應用程式產生的內容作出的回應

用戶可透過多種方式向應用程式產生的內容作出的回應。

內容意見可包括但不限於隱藏或標記動態為垃圾訊息。確保保持串流動態的品質，因為檢視動態的收件人及/或用戶會對串流動態作出評價，而您希望他們對此內容作出正面回應。（請記住，在個人檔案上發佈的內容有可能在發佈用戶的朋友的動態消息上出現。）

至於其他渠道，如相片及活動，意見可包括但不限於取消標籤、刪除或舉報相片；刪除或舉報活動等等。
