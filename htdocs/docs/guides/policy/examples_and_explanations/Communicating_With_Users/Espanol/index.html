Ejemplos y explicaciones: Comunicación con tus usuarios

El canal del conjunto de publicaciones no está diseñado para la comunicación de aplicación a usuario, como las actualizaciones diarias, sino para que los usuarios compartan contenido interesante y relevante con sus amigos. Hemos elaborado productos específicos que permiten a las aplicaciones comunicarse directamente con sus usuarios de una forma adecuada. A continuación se ofrecen ejemplos de herramientas de comunicación "aplicación a usuario".

Solicitudes generadas por aplicaciones

A diferencia de las solicitudes de usuario, las solicitudes generadas por las aplicaciones se pueden iniciar a través de la API de la gráfica social que actualiza el recuento de marcadores. Se trata de una forma excepcional de fomentar la interacción y solicitar a los usuarios que lleven a cabo una acción, como mover una pieza de ajedrez, atender su granja o consultar las actualizaciones diarias, como clasificaciones deportivas, horóscopos y artículos de noticias.

Nota: esta función solo está disponible para las aplicaciones con página principal.

Páginas de las aplicaciones

El uso de la página de la aplicación para comunicarte con tus usuarios constituye una forma excelente de impulsar la participación en tu aplicación y de informar a los usuarios de las novedades de la aplicación, como un nuevo nivel o artículos para comprar. Al igual que en el resto de páginas de Facebook, los administradores pueden comentar las publicaciones de los usuarios, subir fotos e interactuar directamente con los usuarios.

Páginas de la gráfica abierta

Si incluyes etiquetas de gráfica abierta en tu página web esta será equivalente a una página de Facebook. Esto significa que cuando un usuario haga clic en el botón "Me gusta" de tu página, se establecerá una conexión entre tu página y el usuario. Tu página aparecerá en la sección "Gustos e intereses" del perfil del usuario y podrás publicar actualizaciones para el usuario que se mostrarán en la sección "Últimas noticias".

Correo electrónico

El correo electrónico es el mejor método para la comunicación de aplicación a usuario. Puedes solicitar la dirección del usuario a través del permiso adicional de correo electrónico o pedirle directamente que te facilite su dirección proporcionando un campo específico para ello.
