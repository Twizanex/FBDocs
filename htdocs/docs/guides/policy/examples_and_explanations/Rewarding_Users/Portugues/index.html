Exemplos e explicações: Recompensa a usuários

Recompensas com base em referências

Apesar de nossas políticas proibirem a criação de laços diretos por meio de incentivos ao uso de nossos canais sociais, por exemplo, recompensas a usuários pela simples publicação de históricos de fluxo ou pelo envio de uma solicitação, permitimos recompensas com base em referências, nas quais nossos canais sociais são indiretamente ligados á possível recompensa no aplicativo. Por exemplo, você pode recompensar usuários com base no número de amigos que aceitam o convite do usuário para usar o aplicativo. No exemplo a seguir, observe que o usuário não recebe uma recompensa por enviar o convite, mas tem a chance de receber recompensas caso tenha amigos que aceitem o convite.

Locais

Aplicativos podem incentivar usuários a fazer check-in em locais por meio de descontos, cupons e, por exemplo, entradas adicionais para uma promoção para usuários que fizerem check-in no seu local. Da mesma forma que com as funcionalidades nativas do Facebook, o check-in não deve ser usado como um meio de incluir ou registrar usuários na promoção.

Botão Curtir - Recompensa a fãs

Também permitimos recompensas específicas relacionadas ao botão Curtir, contanto que o incentivo esteja aberto a todos os usuários novos e existentes que curtirem sua página.

Algumas recompensas comuns a todos os usuários que curtem sua página incluem:

  • Cupons/Descontos
  • Conteúdo exclusivo
  • Elegibilidade para participar de uma promoção
  • Doação para caridade com base no número de opções Curtir da sua página
