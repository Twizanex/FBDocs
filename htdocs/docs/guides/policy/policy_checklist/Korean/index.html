플랫폼 정책 및 홍보 체크리스트

Date of Last Revision: August 10, 2011

이 체크리스트는 Facebook 플랫폼 정책이나 홍보 가이드라인의 전체 리스트가 아니며, 앱을 런칭하기 전 앱의 기능과 활용 방법에 대해 추가적인 가이드라인을 제공하기 위한 것입니다.

개인 정보

앱은 캔버스나 웹사이트상 또는 개발자 앱 개인 정보 필드 내에 자체의 개인 정보 URL을 포함하여 Facebook이 데이터 허가 대화 상에서 개인 정보 정책으로 가는 링크를 제공할 수 있도록 합니다.

데이터와 콘텐츠

앱은 어떠한 광고 크리에이티브 상에서도 사용자와 관련해 Facebook으로부터 받은 데이터를 포함하지 않습니다.

앱의 광고는 좋아요 버튼과 같은 소셜 플러그인을 포함한 어떠한 플랫폼 통합도 포함하지 않습니다.

소셜 채널

앱은 광고나 Facebook 소셜 채널을 통해 배포된 콘텐츠 내에서 다른 앱과의 공동 홍보를 포함하지 않습니다.

앱은 사용자들에게 스트림 스토리, 요청과 같은 Facebook 소셜 채널 사용을 직접적으로 요구하거나 이에 대해 보상을 제공하지 않습니다.

앱은 사용자가 사전에 콘텐츠를 직접 생성하지 않은 한 앱의 스트림 스토리나 사진 설명의 사용자 메시지 섹션을 미리 채우지 않습니다.

앱은 사용자를 대신해 스트림 스토리를 자동으로 게시하지 않으며, 대신 콘텐츠가 공유된다는 사실을 사용자가 명확히 인지할 수 있도록 버튼을 클릭하거나 박스에 체크할 수 있는 옵션을 제공합니다.

브랜딩 및 기능

앱은 Facebook의 상표를 사용하거나 Facebook과의 제휴 관계, Facebook으로부터의 지원을 암시하는 콘텐츠를 게시하지 않습니다.

앱은 Facebook의 서비스를 폄하하는 개념이나 기능을 포함하지 않습니다.
(예: 사용자들에게 친구나 연결 관계를 삭제할 것을 권하지 마십시오. 사용자의 동의가 있다 하더라도 사용자의 프로필을 넘겨받지 마십시오. 사용자들에게 실제 인물이 아닌 사람을 태그하라고 권하지 마십시오. 사용자들에게 친구 삭제 사실을 알리지 마십시오. 사용자의 개인 정보 설정을 존중해 주십시오.)

홍보 가이드라인

Facebook 상의 홍보 행사는 모든 응모자나 참가자가 해당 행사에 Facebook이 전혀 관여하지 않음을 인지하는 내용과 함께 참가자가 관련 정보를 Facebook이 아닌 정보 접수하는 당사자에게 제공하며 해당 행사가 어떤 식으로도 Facebook으로부터 지원, 관리, 참여를 받지 않음을 인정하는 내용을 게시합니다.

홍보 행사는 등록/참가 과정에서 Facebook의 기능을 이용하지 않습니다. 참고: 참가란 홍보 행사(앱이 아닌)에의 참가를 말합니다.

홍보 행사는 가입을 전제로 하거나 사용자들의 페이지에 대한 좋아요 설정, 장소에 체크인, 앱과의 연결을 제외한 다른 Facebook의 기능을 사용해 행동을 취함을 전제로 참가를 허용하지 않습니다.

홍보 행사는 좋아요 버튼과 같은 Facebook의 기능을 투표 수단으로 사용하지 않습니다.

홍보 행사는 당첨자에게 Facebook 메시지, 채팅, 프로필이나 페이지 게시물 등의 Facebook 기능을 이용해 당첨 사실을 알리지 않습니다.
