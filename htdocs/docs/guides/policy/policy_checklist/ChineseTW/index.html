平台政策與推廣指南注意事項

Date of Last Revision: August 10, 2011

本表並非 Facebook 開放平台政策或推廣指南的完整列表，目的在於提供開發人員額外的指導參考，協助其審核程式上線前功能與特性應注意事項。

隱私

在應用程式存在的 canvas 或網站以及基本資料設定的隱私欄位輸入應用程式隱私條款網址(URL)，以便Facebook在數據授權對話視窗中提供相對應的隱私條款連結。

數據資料與內容

應用程式並未在任何廣告創意中包含從 Facebook 處取得的用戶數據資料。

應用程式的廣告並未含有任何的平台集成元素，如安裝「讚」鈕等社交外掛。

社交途徑

應用程式經由 Facebook 各個社交途徑所發佈的訊息中不含廣告或交叉推廣其他應用程式的內容。

應用程式並未直接獎勵或要求用戶使用 Facebook 的任何社交途徑，如發佈動態消息以及發送邀請等等。

應用程式並未預先填寫動態消息或相片說明的 user_message 參數內容，除非該內容是用戶已經在先前的流程中手動輸入產生。

應用程式不會自動地以用戶的名義發佈動態消息，而是提供用戶以點擊按鈕或勾選方框的選擇方式取得他們的同意，清楚說明在他們點擊或勾選後其內容將會被發佈分享。

品牌與功能

應用程式並未使用 Facebook 商標，或以明示或暗示的方式，表示該應用程式與 Facebook 之間存在任何關聯或具有 Facebook 的認可背書。

應用程式中並未包含破壞 Facebook 產品方針的概念或功能。
(如：不鼓勵用戶刪除朋友關係或關係鏈；即使用戶同意，也不接管用戶的個人檔案；不鼓勵用戶張貼除了真人以外的姓名標籤；不就朋友關係的解除進行任何通知；務必尊重用戶的隱私設定。)

推廣指南

在 Facebook 上進行的推廣宣傳必須包含下列原則：取得每位參賽者或參加者對 Facebook 的完全免責權的同意；明列推廣活動與 Facebook 無關，並非由 Facebook 贊助、支持或管理的文字說明；參加者提供資料的對象為 [公開資料的接收者] 而非 Facebook 的公開說明。

推廣活動不可利用 Facebook 功能作為參與活動的登記或參加機制。 注意：參與的定義為加入推廣活動，而非應用程式的使用。

您不可要求用戶使用任何 Facebook 功能執行特定動作（除了到粉絲專頁按讚、至特定地點簽到打卡或連結至應用程式以外）作為註冊或參加活動的條件。

推廣活動不可利用 Facebook 功能(如讚鈕)作為參與活動的票選機制。

推廣活動不可透過 Facebook 功能通知得獎、優勝者，例如不可利用 Facebook 訊息、聊天、或在個人檔案或頁面上公佈優勝者名單訊息等。
