FacebookApiException::getType()

Get the type for the error or exception, e.g. OAuthException.

Example

try {
  $facebook->api('/me/feed','POST',
                   array( 
                     'message' => 'Hello World!',
                     'link' => 'www.example.com'
                        )
                );
} catch(FacebookApiException $e) {
  $e_type = $e->getType();
  error_log('Got an ' . $e_type . ' while posting');
}

Parameters

None.
