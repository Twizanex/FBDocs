FB.Canvas.startTimer

SDK Reference › JavaScript SDK › FB.Canvas.startTimer

To be used in conjunction with setDoneLoading . Call startTimer to resume the timer after a period of time for the page load that you didn't wish to measure, which you began by calling stopTimer.

Later, call setDoneLoading when the page load is complete.

Examples

FB.Canvas.startTimer();
