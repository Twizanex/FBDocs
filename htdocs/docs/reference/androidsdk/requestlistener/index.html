RequestListener

SDK Reference › Android SDK › RequestListener

The RequestListener Interface provides the callback methods for asynchronous request methods. These are defined in AsyncFacebookRunner.java.

Callback Methods

void onComplete(String response, Object state):

Called when a request completes with the given response. Executed by a background thread: do not update the UI in this method.

Parameters

response - JSON string representation of the API request response.

state - An arbitrary object used to identify the request that was passed in the original request method.

void onFacebookError(FacebookError fbe, Object state):

Called when the server-side Facebook method fails. Executed by a background thread: do not update the UI in this method.

Parameters

fbe - Encapsulation of a Facebook Error, a Facebook request that could not be fulfilled. This is defined in FacebookError.java

state - An arbitrary object used to identify the request that was passed in the original request method.

void onFileNotFoundException(FileNotFoundException ex, Object state):

Called when a request fails because the requested resource is invalid or does not exist. Executed by a background thread: do not update the UI in this method.

Parameters

ex - FileNotFoundException pertaining to the invalid or not found resource.

state - An arbitrary object used to identify the request that was passed in the original request method.

void onIOException(IOException ex, Object state):

Called when a request has a network or request error. Executed by a background thread: do not update the UI in this method.

Parameters

ex - Network error IOException.

state - An arbitrary object used to identify the request that was passed in the original request method.

void onMalformedURLException(MalformedURLException ex, Object state):

Called if an invalid graph path is provided (which may result in a malformed URL). Executed by a background thread: do not update the UI in this method.

Parameters

ex - MalformedURLException pertaining to the invalid graph path.

state - An arbitrary object used to identify the request that was passed in the original request method.
