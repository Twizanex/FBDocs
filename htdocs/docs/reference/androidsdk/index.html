Android SDK Reference

SDK Reference › Android SDK Reference

The Android SDK provides Facebook Platform support for Android apps. It enables you to implement frictionless Single Sign On and access the Facebook Platform APIs including the Graph API, FQL, and Dialogs.

The SDK is open source and is available on GitHub. Read more in Android Tutorial on how to set up the Android SDK.
