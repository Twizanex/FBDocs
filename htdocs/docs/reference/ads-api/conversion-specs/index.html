Conversion Specs

Conversion specs allow advertisers to define conversion objectives for their Ad groups based on meaningful Facebook actions such as likes on a page post or installs of an app. If you use conversion specs for an ad, you can not only measure the delivery of your ads against these objectives, but also optimize your ads towards these objectives using Optimized CPM.

For example, the following conversion spec measures number of comments that a page post received from users who clicked on an ad or a sponsored story.

{'action.type':'comment', 'post':1234567}

Where 1234567 is the post ID.

Conversion specs are available for on Facebook actions (listed below) as well as global open graph actions (listen, watch, and read) and custom open graph actions.

Defining Conversion Specs

To create an ad group that uses a conversion spec, use a HTTP POST request like below example. You can specify one or more conversion specs using the conversion_specs parameter.

curl
 -F "name=Test AdGroup"
 -F "conversion_specs={'action.type':'like', 'post.wall':133445987}"
 -F "campaign_id=1243234444"
 -F "bid_type=1"
 -F "max_bid=30"
 -F "targeting={'countries':['US']}"
 -F "creative={'type':27, 'object_id':133445987, 'auto_update':true}"
     "https://graph.facebook.com/act_1234567/adgroups?access_token=___"

The response will be the standard Ad group creation response.

If you want to track more than one action, you can set conversion_specs parameter to a JSON array like below example.

curl
 -F "name=Test AdGroup"
 -F "conversion_specs=[{'action.type':'like', 'post.wall':133445987},{'action.type':'comment', 'post.wall':133445987}]"
 -F "campaign_id=1243234444"
 -F "bid_type=1"
 -F "max_bid=30"
 -F "targeting={'countries':['US']}"
 -F "creative={'type':27, 'object_id':133445987, 'auto_update':true}"
     "https://graph.facebook.com/act_1234567/adgroups?access_token=___"

The response will be the standard Ad group creation response.

Note: Conversion specs can be configured for any Ad group, even if the Ad Creative does not refer to the same object the conversion spec does. E.g. A Page Ad for Page ID 123456 could have a conversion spec tracking app installs for App ID 456789, even if the page and app were unrelated. The only requirement is that the objects in the conversion specs need to belong to the Ad Group creator, (i.e. that they are an admin of the object).

Meta Specs

Meta Specs allow you to track multiple types of actions on a single object using a single spec.

Object Conversion Spec Description
Page {'action.type':'page_story', 'page':PAGE_ID} Number of times users generate stories about a page via like, checkin, comment, post (+share), receive_coupon, vote, follow, mention for specific page
Page {'action.type':'page_engagement', 'page':PAGE_ID} Number of times users generate stories (page_story) or engage with a page via link_click, photo_view, video_play.
Page Post {'action.type':'post_story', 'post':POST_ID, 'page':PAGE_ID} Number of times users generate stories about a page post via like, comment, share, receive_coupon, vote, follow for a specific page post
Page Post {'action.type':'post_engagement', 'post':POST_ID, 'page':PAGE_ID} Number of times users generate stories (post_story) or engage with a page post via link_click, photo_view, video_play
Application {'action.type':'app_story', 'application':APPLICATION_ID} Number of times users generate stories about an app via app_use
Application {'action.type':'app_engagement', 'application':APPLICATION_ID} Number of times users generate stories (app_story) or engage with content via app_use, app_install, credit_spent.

Default Conversion Specs

If no conversion specs are applied to a Page, App or Event Ad group then default conversion specs are applied as follows:

Object Conversion Spec Description
Page {'action.type':'page_engagement', 'page':PAGE_ID} Number of times users generate stories (page_story) or engage with a page via link_click, photo_view, or video_play
App {'action.type':'app_engagement', 'application':APPLICATION_ID} Number of times users engage stories (app_story) or engage with content via app_use, app_install, or credit_spent.
Event {'action.type':'rsvp', 'event': EVENT_ID} Number of RSVPs (yes and maybe) to an event.

Conversion specs for Custom Open Graph Actions

Both global custom open graph actions (watch, listen, and read) and custom open graph actions can be used as conversion specs. For example, you can use the following conversion spec for tracking the number of additional listens that resulted from viewing an ad.

{'action.type': 'music.listens', 'application': 12398755}

Where 12398755 is the ID of the application generating the music.listens actions.
