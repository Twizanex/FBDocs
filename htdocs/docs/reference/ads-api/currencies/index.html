Offset

Each currency has an offset which explains how its sub-divisions are dealt with in the Ads System. This is to ensure that the minimum bid (i.e. "1") is a usable value for the currency.

Example offset 100

If a currency has an offset of 100 then the minimum allowed bid (i.e. max_bid = 1) is assumed to be 1/100 of the currency base unit. E.g. If you place a bid of "1" on a USD based Ad Account the bid will be 0.01 USD.

Example offset 1

If a currency has an offset of 1 then the minimum allowed bid (i.e. max_bid = 1) is assumed to be in the currency base unit. E.g. If you place a bid of "1" on a JPY based Ad Account the bid will be 1 JPY.
