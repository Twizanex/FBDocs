Keyword Stats

Advanced Topics › Ads API › Keyword Stats

Keyword stats are stats specific to each keyword used in an Ad group's targeting spec. Keyword stats can be queried for the past 7 days (excluding the current day)

Note: The IDs specified in adgroup_ids must be for ad groups active in the past 7 days. Additionally, the ad groups must use keyword targeting.

Note: Keywords are referred to as "Interests" in the Ad Manager UI.

Query Result Fields

Name Description Permissions Returns
impressions The total number of times your adverts have been shown on the site. ads_management int
clicks The number of clicks the Sponsored Story or advert has received. If you're promoting a Page, event or application, clicks will also include Page likes, event RSVPs or installation of applications directly from the Sponsored Story or advert. ads_management int
unique_impressions The number of individuals this ad was shown to on the site ads_management int
unique_clicks The number of individuals who clicked this ad. ads_management int

Querying the keyword stats of an adgroup

To query the keyword stats of an ad group, query the keywordstats connection of the Ad group object:

{adgroup_id}/keywordstats
