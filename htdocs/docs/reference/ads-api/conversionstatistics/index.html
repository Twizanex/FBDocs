Conversion Stats

Advanced Topics › Ads API › Conversion Stats

Conversion stats can be retrieved at Ad group, Ad campaign and Ad account levels. Each conversion stat returns six attribution values for each conversion that is being tracked. These attributions are defined as follow.

Attribution Value Description
post_imp_1d Number of conversions within 1 day after viewing an ad
post_imp_7d Number of conversions within 7 days after viewing an ad
post_imp_28d Number of conversions within 28 days after viewing an ad
post_click_1d Number of conversions within 1 day after clicking on an ad
post_click_7d Number of conversions within 7 days after clicking on an ad
post_click_28d Number of conversions within 28 days after clicking on an ad

Note: Conversion stats update in near real-time.

Below examples show how to fetch conversion reports for each object.

Ad Account Conversion Stats

To retrieve conversion stats for an Ad account use the following request:

curl https://graph.facebook.com/act_<ACT_ID>/conversions?access_token=__

Ad Campaign Conversion Stats

To retrieve conversion stats for all Ad campaigns under an Ad account use the following request:

curl https://graph.facebook.com/act_<ACT_ID>/adcampaignconversions?access_token=__

To retrieve stats for specific Ad campaigns within an Ad account, use the campaign_ids parameter as show in the following request:

curl https://graph.facebook.com/act_<ACT_ID>/adcampaignconversions?campaign_ids=[123456,1258998]&access_token=__

Alternatively to retrieve stats for a single Ad campaign, use the conversion connection as show in the following request:

curl https://graph.facebook.com/123456/conversions?access_token=__

Where 123456 is an Ad campaign ID.

Ad Group Conversion Stats

To retrieve conversion stats for all Ad groups in an Ad account use the following request:

curl https://graph.facebook.com/act_<ACT_ID>/adgroupconversions?access_token=__

To retrieve stats for specific Ad groups within an Ad account, use the adgroup_ids parameter as show in the following request:

curl https://graph.facebook.com/act_<ACT_ID>/adgroupconversions?adgroup_ids=[22113345,980987]&access_token=__

Alternatively to retrieve stats for a single Ad group, use the conversion connection as show in the following request:

Additional Parameters

The following additional parameters are available on Conversion Stats requests.

Time Range

To fetch conversion stats for a specific date range, append start_time and end_time in date format.

curl https://graph.facebook.com/act_<ACC_ID>/adcampaignconversions?start_time=2012-01-20&end_time=2012-02-05&adcampaign_ids=[666666]&access_token=__

Alternatively, for UNIX timestamp format, use time_start and time_stop parameters.

curl https://graph.facebook.com/act_21002924/adcampaignconversions?time_start=1327045120&time_stop=1328427520&adcampaign_ids=[666666]&access_token=_

Aggregation Window

When you use a time range, you may want to specify an aggregation window. If you set the aggregation window to 7 days, it will aggregate conversions for the past 7 days ending on each day in the time range. The default aggregation window is 0 days, that is, the total number of conversions in the time range. To specify an aggregation window, set aggregate_days parameter. For example, the following request asks for conversion stats for each day between 01/20/2012 and 12/05/2012. The conversion stats for each day is aggregated based on 5 day windows ending at that day.

curl https://graph.facebook.com/act_21002924/adcampaignconversions?aggregate_days=5&start_time=2012-01-20&end_time=2012-02-05&adcampaign_ids=[666666]&access_token=__

Note: The maximum permitted aggregate_days is 90 days.

Default Request Parameters for Lifetime Conversion Stats

The default values for start_time, end_time, and aggregate_days are 0, that is the API will return the lifetime stats of this Ad group / Ad campaign/ Ad account if above parameters are set to zero (or not specified in the request).

Aggregation Type

You can aggregate conversion stats based on conversion time (default) or impression time. To aggregate stats based on click time, set by_impression_time parameter to false.

curl https://graph.facebook.com/act_21002924/adcampaignconversions?by_impression_time=false&adcampaign_ids=[666666]&access_token=__
  • A person's app used Featured story is eligible to be shown to his/her friends if he/she used the app for at least 4 minutes in the last 14 days or used the app twice ever with one of those times occurring in the last 14 days.

Limit

The limit is ONLY on the number of entries directly under the "data" field, each of which is the conversion stats of an Ad group, Ad campaign or Ad account with a certain starting date. Within each entry, conversions are broken down by the "action_type" and "object_id".
