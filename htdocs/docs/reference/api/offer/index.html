Offer

Core Concepts › Graph API › Offer

Offers is currently in beta and is available to a limited set of Pages in New Zealand, Singapore, Australia, Japan, and Turkey. We plan to launch offers more widely in the next few weeks. To learn more, please see our recent blog post.

An Offer represents an offer that is published by a page. Only Page objects have offers connections.

To read offers, you need one of the following:

  • A valid Page Access token
  • manage_pages permission

To read an offer object, issue an HTTP GET request to /OFFER_ID with the correct permission.
