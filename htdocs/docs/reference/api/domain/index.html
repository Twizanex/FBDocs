Domain

Core Concepts › Graph API › Domain

A web site domain within the Graph API. To register your own Domain, you must claim your domain name using Facebook Insights.

Example

https://graph.facebook.com/369296215699
https://graph.facebook.com?domain=www.facebook.com
https://graph.facebook.com?domains=www.facebook.com,www.example.com

Fields

Name Description Permissions Returns
id

The ID of the domain

No access_token required

string

name

The name of the domain

No access_token required

string
