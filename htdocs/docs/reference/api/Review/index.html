Review

Core Concepts › Graph API › Review

An object representing a review for an application.

The Application object has a reviews connection containing review objects.

Example

Getting the reviews of the App Dashboard:

https://graph.facebook.com/2345053339/reviews

Fields

The review object has the following fields.

Name Description Permissions Returns
id

The Facebook ID of the review

No access_token required

string

from

The user that created the review

No access_token required

object containing the id and name of the creating user
to

The application to which this review applies

No access_token required

object containing the id and name of the application
message

The review text (optional)

No access_token required

string

rating

The review rating

No access_token required

int between 1 and 5
created_time

The timedate the review was created

No access_token required

string containing ISO-8601 date-time
