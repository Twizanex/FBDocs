offer

Advanced Topics › FQL › offer

Offers is currently in beta and is available to a limited set of Pages in New Zealand, Singapore, Australia, Japan, and Turkey. We plan to launch offers more widely in the next few weeks. To learn more, please see our recent blog post.

An FQL table that returns information about offers that a Page posts.

To read offer table you need:

  • the manage_pages permission
  • A Page access token

Example

Retrieve the recent offers for a page:

SELECT id, title FROM offer WHERE owner_id = PAGE_ID

コラム

インデックス可能 名前 タイプ 説明
* id int

The ID of the offer.
* owner_id int

The ID of the Page that this offer belongs to.
title string

The title of the offer.
image_url string

The URL of the image for the offer.
terms string

Text description of the terms under which the offer can be claimed.
claim_limit int

The maximum number of times the offer can be claimed.
created_time int

The UNIX timestamp when the offer was created
expiration_time string

The UNIX timestamp when the offer expires (this is only for display purposes).
