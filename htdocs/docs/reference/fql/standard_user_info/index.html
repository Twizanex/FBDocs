standard_user_info

Advanced Topics › FQL › standard_user_info

An FQL table which returns basic information about a user. Note: For use only for analytic information.

Note: Don't display this information to any users. If you need to display information to other users, query the user FQL table instead.

You cannot query this table for a desktop application.

To read the standard_user_info table you need

  • an app access_token to get information about any authorized app users

Example

Retrieve info about the user, Bret Taylor:

SELECT name, locale FROM standard_user_info WHERE uid = 220439

コラム

インデックス可能 名前 タイプ 説明
* uid string

The user ID of the user being queried.
* name string

The full name of the user.
* username string

The username of the user.
* third_party_id string

Third party ID of the user.
first_name string

The first name of the user.
last_name string

The last name of the user.
locale string

The two-letter country code for the user's locale. Codes used are the ISO 3166 alpha 2 code list.
affiliations array

The corporate or educational networks to which the user being queried belongs. Regional networks have been deprecated and are no longer returned.
profile_url string

The URL to a user's profile.
timezone string

The time zone where the user being queried is located.
birthday string

The birthday of the user being queried.
sex string

The sex of the user.
proxied_email string

The proxied wrapper for a user's email address.
current_location string

The city in which the user currently lives.
allowed_restrictions string

A comma delimited list of Demographic Restrictions types a user is allowed to access. Currently, alcohol is the only type that can be returned.
