page

Advanced Topics › FQL › page

An FQL table that can be used to return information about a Facebook Page.

The Page Graph API object provides similar information to this table.

To read the page table you need

  • no access_token to get public information, or information about public pages that are not demographically restricted.
  • any valid access_token to get all information about a page that the current session user is able to see.

Example

Public information about the Facebook Platform Page (Try this Query):

SELECT name, fan_count FROM page WHERE page_id = 19292868552

Information about the Engineering Page retrieved by querying against its URL. (Try this Query)

SELECT page_id, pic FROM page WHERE username = "Engineering"
