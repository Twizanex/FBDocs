stream_tag

Advanced Topics › FQL › stream_tag

An FQL table that shows the association between Users or Pages and objects they tag.

To read the stream_tag table you need

  • any valid access_token if the post containing the tag is public.
  • read_stream permissions for all other posts visible to the current session user.

Examples

Get all the posts in which a particular user has tagged someone or something on Facebook (Try this query):

SELECT post_id,target_id FROM stream_tag WHERE actor_id=563683308

Get all the posts where a specific user was tagged, and which friends tagged that user (Try this query):

SELECT post_id,actor_id FROM stream_tag WHERE target_id=563683308

Get all the users tagged in a given post (Try this query):

SELECT target_id FROM stream_tag WHERE post_id=10150207376128309

コラム

インデックス可能 名前 タイプ 説明
* post_id string

The ID of a post from the user's or Page's stream. This field, when used as an index, is used to retrieve users and the entities they've tagged.
* actor_id string

The user ID of the user or Page who tagged one or more entities in a post. This field, when used as an index, is used to retrieve all the posts and the entities the user has tagged.
* target_id string

The users, Pages, events, and other Facebook objects that have been tagged in one or more posts. This field, when used as an index, is used to retrieve all the entities that have been tagged in one or more posts.
