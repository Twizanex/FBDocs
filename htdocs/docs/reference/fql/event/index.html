event

Advanced Topics › FQL › event

An event which users can attend as represented in FQL.

The User and Page objects have an equivalent events connection which gives a summary of associated events. You can access additional information on these events by querying them by ID, which will return an object of type Event.

To read the event table you need:

  • no access_token is required if the event is public.
  • a valid user access_token for a user who can see the event if it has restricted visibility
  • an application access_token for your app if your app created the event

Starting July 5th, an access token will be required to access even public events. To read the event table, you will need one of the following:

  • a user access token for a user who can see the event
  • an app access token (for non-public events, must be the app that created the event)
  • a page access token (for non-public events, must be the page that created the event)

Example

An event on the Facebook Engineering Page:

The "Pushing millions of lines of code five days a week tech talk" event (Try this query)

SELECT name, venue, location, start_time FROM event WHERE eid = 209798352393506

* eid int

The ID of the event being queried.
name string

The name of the event being queried.
tagline string

The tagline or summary of the event being queried. Deprecated - This column will be removed on July 5th, 2012.
nid int

The network ID of the event being queried. Deprecated - This column will be removed on July 5th, 2012.
pic_small string

The URL to the small-sized profile picture for the event being queried. The image can have a maximum width of 50px and a maximum height of 150px. This URL may be blank.
pic_big string

The URL to the largest-sized profile picture for the event being queried. The image can have a maximum width of 200px and a maximum height of 600px. This URL may be blank.
pic_square string

The URL to the square-sized profile picture for the event being queried. The image is returned with a width and height of 50px.
pic string

The URL to the medium-sized profile picture for the event being queried. The image can have a maximum width of 100px and a maximum height of 300px. This URL may be blank.
host string

The name of the host of the event being queried.
description string

The description of the event being queried.
event_type string

The category of the event being queried. Deprecated - This column will be removed on July 5th, 2012.
event_subtype string

The event type for the event being queried. Deprecated - This column will be removed on July 5th, 2012.
start_time time

The starting date and time of the event being queried.
end_time time

The ending date and time of the event being queried.
creator int

The user ID of the creator of the event being queried.
update_time time

The time that the event being queried was last updated.
location string

The location of the event being queried.
venue array

The venue where the event being queried is being held. Contains one or more of the following fields: street, city, state, zip, country, latitude, and longitude.
privacy string

The privacy setting of the event being queried, indicating whether the event is OPEN, CLOSED, or SECRET.
hide_guest_list bool

Indicates whether the guest list on the event's page is hidden..
can_invite_friends bool

Indicates whether or not the viewer can invite friends to the event.
