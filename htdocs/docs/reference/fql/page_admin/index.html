page_admin

Advanced Topics › FQL › page_admin

An FQL table containing information about the pages that a user is an admin of.

The User Graph API object has a connection accounts which contains similar information.

To read the page_admin table you need

  • manage_pages permissions for the list of pages the current session user is an admin of.

Note: We previously only required any valid access_token to query this table, however from September 22, 2011 we will be requiring the above permission. To avoid breaking changes, please start using that permission before then.

Example

The list of pages that the current session user is an admin of (Try this query):

SELECT page_id, type from page_admin WHERE uid=me()
