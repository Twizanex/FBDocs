users.isVerified

Advanced Topics › Legacy REST API › users.isVerified
Please note: We are in the process of deprecating the REST API. If you are building a new application use the Graph API and access the verified field of the User object

Determines whether Facebook considers the current user to be verified or not. We use verification with certain features such as granting verified users increased limits for the length and file size of videos they can upload directly on Facebook or [your applications](through)(video.upload).

Important: You may never disclose a user's verified state anywhere in your application or on Facebook, whether to that user or the user's friends.

パラメータ

名前 タイプ 説明
callback string

Name of a function to call. This is primarily to enable cross-domain JavaScript requests using the <script> tag, sometimes known as "JSONP". This works with both XML and JSON.

レスポンス

This call returns true or 1 for a verified user; false or 0 for an non-verified user.
