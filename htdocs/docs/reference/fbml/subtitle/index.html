fb:subtitle

Advanced Topics › Legacy FBML › fb:subtitle
FBML has been deprecated. Starting June 1, 2012 FBML apps will no longer work as all FBML endpoints will be removed. If you are building a new application on Facebook.com, please implement your application using HTML, JavaScript and CSS. You can use our JavaScript SDK and Social Plugins to embedded many of the same social features available in FBML.

Defines the subtitle for the profile box. The enclosed content is interpreted purely as plain text.

This tag may contain fb:action, an optional tag that renders an action link on the right-hand side of the subtitle.

パラメータ

名前 タイプ 説明
seeallurl string

The URL for a "See all" link. This URL must be a facebook canvas page.

導入例

<fb:subtitle seeallurl="http://apps.facebook.com/myapp/seeall" >
      <fb:action href="http://apps.facebook.com/fun_graphics_app/new.php">Post a new fun graphic</fb:action>
      Displaying 10 images 
 </fb:subtitle>
