Facebook.showProfileTabDialog

Description

This FBJS call renders an Add Profile Tab button on your canvas or Web page, enabling the user to add your application tab to his or her profile.

If the user already added your application tab, the button doesn't appear.

You can determine if a user has added your application tab by querying the tab_added field in the permissions FQL table.

Parameters

Required Name Type Description
callback JS_API_T_System.Callback Callback function to execute after this call completes. It takes one argument, which is a JavaScript object. The object has one key, tab_added, the value for which is 1 or 0.

Example

Facebook.showProfileTabDialog(callback);
